# Paw Print (prototype)

A simple, secure portal for storing pet vaccination records so people experiencing homelessness in Kamloops aren't turned away from shelter or supported housing when they can't produce proof of their animal's vaccinations. Built for Same Sky Collective Society & Four Paws.

Live demo: https://zachrobins.github.io/paw-print-demo/

Prototype only — data lives in the browser; next step is a Canada-hosted backend.
